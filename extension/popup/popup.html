How it works:

  1. Navigate to any sub-account page in Zoom admin
  2. Click Record below
  3. Perform the workflow once (fill forms, click buttons)
  4. Stop when done — account-specific values are auto-detected
  5. Export → the workflow runs on any sub-account automatically

💡 You don't need to record login or account switching — the automation engine handles impersonation for each sub-account automatically.
